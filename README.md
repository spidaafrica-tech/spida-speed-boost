# SPIDA Africa - Agricultural Technology Platform

## About SPIDA

SPIDA (Smart Platform for Integrated Digital Agriculture) is transforming Africa's agricultural landscape through technology and innovation. We provide comprehensive solutions that address every aspect of the agricultural value chain, from mechanization and financing to market access and logistics.

## Our Mission

To create an inclusive, digitally connected agricultural ecosystem that ensures every farmer thrives with higher productivity, guaranteed market access, and sustainable farming practices.

## Technology Stack

This project is built with modern web technologies:

- **Vite** - Fast build tool and development server
- **React 18** - Modern UI library with TypeScript
- **TypeScript** - Type-safe JavaScript development
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Beautiful, accessible UI components
- **React Router** - Client-side routing
- **Lucide React** - Beautiful icon library

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd spida-speed-boost
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:8080` to view the application.

## Available Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build for production
- `npm run build:dev` - Build in development mode
- `npm run preview` - Preview the production build
- `npm run lint` - Run ESLint for code quality
